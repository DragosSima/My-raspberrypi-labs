# My-raspberrypi-labs
Simulated enterprise network on Raspberry Pi 5 to practice CompTIA Security+ and Pentest+ skills.
## Documentation  
All labs are documented step-by-step on GitBook:  
https://dragos-sima.gitbook.io/security+-and-pentest+-notes/

## Labs  
- SSH Tunnel Setup  
- IP Static Configuration  
- User and Session Verification  
- ACL and Permissions  
- Domain Controller Simulation  
- Network Segmentation  
- Security and Pentest Scenarios

## Usage  
Clone the repository:  
git clone https://github.com/tuo-username/My-raspberrypi-labs.git  

Explore the `labs/` folder for configurations and the `docs/images/` folder for screenshots.
